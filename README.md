# Milhano School Admissions Dashboard

Dashboard estático de admisiones con las views de Resumen, Seguimiento, Embudo, Adquisición y EOD.

## Publicar en Vercel

1. Sube a la raíz del repositorio: `index.html`, `styles.css`, `app.js`, `seed-data.js` y `vercel.json`.
2. En Vercel importa el repositorio de GitHub.
3. Usa **Framework Preset: Other**.
4. Deja Build Command e Install Command vacíos.
5. Usa `.` como Output Directory únicamente si Vercel solicita un valor.
6. Presiona **Deploy**.

## EOD

La captura actual sirve para demostrar y probar el flujo. Los registros se conservan en el navegador usado para la prueba y pueden exportarse a CSV. Se requiere una siguiente versión para almacenar y compartir los EOD entre Cinthia y Paty.
